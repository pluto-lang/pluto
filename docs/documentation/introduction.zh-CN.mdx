# Pluto

Pluto 是一款致力于帮助开发者**构建云与 AI 应用**的研发工具，解决 AI 应用与开源模型难部署、云的上手门槛高等问题。

Pluto 支持使用 **Python、TypeScript** 等熟悉的编程语言编写应用，**在应用代码中直接定义与使用应用所需的云资源**，如 AWS SageMaker、DynamoDB 等。Pluto 会通过**静态分析**的方式从代码中自动获取应用程序的基础设施资源需求，并在指定云平台（AWS、Kubernetes、阿里云等）上创建相应的资源实例，**简化资源创建和应用部署流程**。
